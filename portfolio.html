About

Great code is like ambrosia; poor code is like quicksand. With great code what you desire actually takes place; poor code results in a slow, unanticipated, agonizing death. My innate curiosity to find the bugs enables me to produce the ambrosia of great code. My process is to visualize the model, develop components, and verify each part functions before assembling the units. I enjoy collaboration with the people responsible for all the application components. My curiosity of people makes me interested in the client's perspective. Great code develops dynamic web applications that work for the client.

Specialties: HTML5, CSS3, Angular, Mongo DB, Javascript 6. Node,JS, Express.JS.
